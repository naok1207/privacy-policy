My Start Up Page プライバシーポリシー

基本方針

本プライバシーポリシーは、naok1207（以下「当開発者」といいます）が提供する Google Chrome 拡張機能「My Start Up Page」（以下「本拡張機能」といいます）において、ユーザーの情報をどのように取扱うかについて定めるものです。当開発者はユーザーのプライバシーを重視し、関連法令を遵守してユーザー情報の適切な管理に努めます。

取得する情報と保存方法

本拡張機能は、ユーザーのブラウザ新規タブのホーム画面上でメモやタスクの機能を提供するにあたり、以下の情報をユーザーの端末内に保存します。

  • メモの内容：ユーザーがホーム画面に入力したテキストメモ
  • タスクの内容：ユーザーがホーム画面に入力したタスク項目およびその完了状態

上記の情報は、Chromeブラウザのローカルストレージ機能（storage 権限による保存領域）を用いて端末内に保存されます。保存されたデータはユーザーのデバイス上に留まり、当開発者を含む外部の者が直接取得することはありません。 また、これら以外のユーザー情報を本拡張機能が収集・保存することはありません。ローカルストレージには自動的な保存期間の期限がないため、データはユーザーが手動で削除するか拡張機能をアンインストールしない限り端末内に保持されます。

利用目的

本拡張機能が保存したメモおよびタスクの情報は、ユーザーのブラウザホーム画面上でメモ・タスク管理機能を提供するためにのみ利用されます。ユーザーが入力した内容を保存しておくことで、ブラウザを再起動した際や次回利用時にも以前のメモやタスクが表示され、ユーザーの利便性を向上させることが目的です。

ユーザー登録について

本拡張機能は、利用に際してユーザーにログインやアカウント登録を求めることは 一切ありません。そのため、氏名・メールアドレスなどの個人を特定できる情報を当開発者が収集することはありません。ユーザーはインストール後すぐに拡張機能を利用開始でき、個別のアカウント作成は不要です。

データの外部送信および第三者提供

本拡張機能で保存されたメモ・タスクのデータはすべてユーザーの端末内に留まります。本拡張機能は、保存されたユーザーデータを外部のサーバーに送信することはありません。また、Chromeの同期ストレージやクラウドサービスも使用していないため、ユーザーデータがインターネット経由で外部にアップロードされることもありません。

当開発者がお客様のメモ・タスクの内容にアクセスすることはなく、これらのユーザーデータが当開発者に送信されることもありません。したがって、ユーザーの情報が第三者に提供・共有されることは一切ありません。当開発者はユーザーの情報を第三者に対して提供、販売することも行っておりません。

使用する権限

本拡張機能が Chrome 拡張のマニフェストで要求・利用する権限は「storage」のみです。この権限は、拡張機能がユーザーデータをブラウザ内に保存するために必要なものです。本拡張機能はそれ以外の権限を一切使用しておらず、ブラウザの履歴や他のデータへのアクセス、外部通信などは行いません。
